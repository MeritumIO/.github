# MeritumIO

Meritum is a collection of composable PHP libraries built on the principle of *natural aristoi* — components that earn their place through demonstrated value, not convention. No all-or-nothing frameworks, no hidden opinions: each package does one thing, has hard boundaries, and can be replaced when something better comes along.

[Read the full philosophy →](https://dev.to/georgeff/natural-aristoi-h6j)
